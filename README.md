# XCM Explorer UI

Vanilla JavaScript frontend for the XCM Explorer and [Ocelloids](https://ocelloids.net) analytics APIs.

## Commands

### Install dependencies

```sh
bun install
```

### Lint the code

```sh
bun lint
```

### Start development server

```sh
bun dev
```

### Build for production

```sh
bun build
```

> Output is written to the `dist/` folder.

Enjoy!
